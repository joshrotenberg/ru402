In section 3, we have introduced the concepts and features to work with vectors, which turn Redis Stack (and the Enterprise and Cloud flavors) into a de facto Vector Database. We have also made examples using the Python programming language and the redis-py client library. While it is possible to use the redis-cli command line client to exploit the vector capabilities of Redis Stack, managing large vector embeddings is impractical, and Python is the standard language for machine learning workflows.

In this section, we will introduce the most popular client libraries to rewrite the simple example introduced in section 3. Then you will have the chance to execute the example using node-redis for Node.js, jedis for Java, and StackExchange.Redis for .NET languages (we'll provide the C# example).

We will also present RedisVL, a Python client library for using Redis as a Vector Database with powerful features such as out-of-the-box semantic caching.
